2. ping 8.8.8.8

The ping command sends ICMP Echo Request packets to a target host to test network connectivity.

The IP address 8.8.8.8 belongs to Google Public DNS and is commonly used because it is reliable and almost always reachable.

The objective of running this command is to verify that the Kali Linux VM has internet access and can communicate with external networks.
